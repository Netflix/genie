=== Remove Tag From Command

==== Description

Remove a tag from an existing command.

NOTE: You can't remove the `genie.id:{id}` and `genie.name:{name}` tags. They will just be added back by the system

==== Endpoint

`DELETE /api/v3/commands/{id}/tags/{tag}`
