=== Update an Entire Command

==== Description

Update the metadata about a command. This method does a complete replace of the resource. Usual flow is to
call to get the most recent state then update what you want and call this API with the result.

NOTE: For more fine grained control call the PATCH API.

==== Endpoint

`PUT /api/v3/commands/{id}`
