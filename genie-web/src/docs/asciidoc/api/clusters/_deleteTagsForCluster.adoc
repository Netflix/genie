=== Remove All Tags From Cluster

==== Description

Remove all the tags for an existing cluster

IMPORTANT: The `genie.id:{id}` and `genie.name:{name}` tags will **NOT** be removed by this operation

==== Endpoint

`DELETE /api/v3/clusters/{id}/tags`
