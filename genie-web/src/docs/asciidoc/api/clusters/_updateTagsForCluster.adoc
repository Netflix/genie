=== Update Tags for a Cluster

==== Description

Update the tags for an existing cluster.

IMPORTANT: The `genie.id:{id}` and `genie.name:{name}` tags can't be removed. They will automatically be added back
by the system.

==== Endpoint

`PUT /api/v3/clusters/{id}/tags`
