=== Add Tags to Cluster

==== Description

Add tags to an existing cluster. Stored as a set so all duplicates are ignored.

NOTE: `genie.id:{id}` and `genie.name:{name}` tags are automatically added by the service.

==== Endpoint

`POST /api/v3/clusters/{id}/tags`
